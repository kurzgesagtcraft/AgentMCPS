# VCPToolBox 项目研究与安装报告

我已经完成了对 VCPToolBox 项目的深度研究、架构分析、环境安装以及核心配置修复工作。

## 1. 项目结构与核心架构
VCPToolBox 是一个高度模块化、插件驱动的 **AI 中间层协议栈**。
- **核心逻辑 (Core)**：由 `server.js` 驱动，利用 `Plugin.js` 作为插件加载器。它实现了一套名为 **VCP (Variable & Command Protocol)** 的自定义指令协议，用于 AI、工具与长短期记忆之间的交互。
- **插件系统 (Plugins)**：拥有极其丰富的插件库（位于 `Plugin/` 目录），支持同步、异步及服务型插件，涵盖了从简单的天气查询到复杂的图像/视频生成（ComfyUI, WebUI）。
- **长短期记忆与 RAG**：
    - **KnowledgeBaseManager.js**：核心 RAG 管理器，结合 SQLite 和 Rust 编写的向量检索引擎。
    - **rust-vexus-lite**：提供底层高性能向量运算（NAPI-RS 绑定）。
    - **RAGDiaryPlugin**：实现基于日记本的语义检索与元思考链。
- **前端管理**：`AdminPanel/` 提供了一个基于 Web 的管理界面，用于实时调控 RAG 参数和编辑工具配置。

## 2. 所需 API 与环境配置
项目运行依赖于 `config.env` 文件，主要需要以下 API 支持：
- **聊天 API (Chat API)**：
    - `API_URL` & `API_Key`：支持 OpenAI 格式的端点。
    - `MainModel`：主对话模型。
- **嵌入 API (Embedding API)**：
    - **已修复支持独立配置**：现已支持将嵌入请求重定向至本地 Ollama 或其他端点。
    - **已优化本地部署稳定性**：针对本地 Ollama (如 `nomic-embed-text`) 可能出现的上下文超限错误，已实施“三重防御”机制。
    - `EMBEDDING_API_URL`：例如 `http://localhost:11434` (Ollama)。
    - `WhitelistEmbeddingModel`：例如 `nomic-embed-text:latest`。
    - `VECTORDB_DIMENSION`：需与模型匹配（如 Nomic 为 `768`）。
    - `WhitelistEmbeddingModelMaxToken`：建议设为 `512` 以确保在分词器差异下的绝对稳定性。
- **工具 API**：
    - 搜索：Tavily API。
    - 生成：SiliconFlow (图像/视频)。
    - 平台：Bilibili Cookie (用于 UrlFetch)。

## 3. 安装与修复记录
- **依赖安装**：
    - 已执行 `npm install`：安装了 Node.js 核心及插件依赖。
    - 已执行 `pip install -r requirements.txt`：安装了 Python 辅助工具（如视频处理、ComfyUI 转换）的依赖。
- **核心修复**：
    - **Embedding 重定向**：修改了 `KnowledgeBaseManager.js` 和 `RAGDiaryPlugin.js`，允许嵌入接口与聊天接口分离，完美支持 Ollama 本地部署。
    - **输入超限防御**：在 `EmbeddingUtils.js` 中增加了紧急二次切割逻辑，并将 `MAX_BATCH_ITEMS` 设为 `1`，彻底解决了本地模型因分词器差异导致的 `Context Length Exceeded` 错误。
    - **配置纠错**：修复了 `config.env` 中 `API_URL` 格式错误导致的 `Invalid URL` 报错。
    - **WebSocket 鉴权同步**：同步了 `config.env` 中的 `VCP_Key` 为 `vcp123456`，解决了 VChat 客户端连接 VCPLog 时的 `socket hang up` (1006) 鉴权失败问题。
    - **缺失文件**：创建了必要的 `semantic_groups.json` 基础文件以确保插件正常初始化。
    - **Rust 组件**：确认 `midi_quantizer.node` 等核心组件已就绪，无需重新编译。

## 4. 启动指南
环境已完全准备就绪。您可以通过以下步骤启动：
1. 确保本地 Ollama 已启动并加载 `nomic-embed-text:latest` 模型。
